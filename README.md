# Trabajo Práctico — Programación Orientada a Objetos

## 📌 Descripción

Este proyecto fue desarrollado como trabajo práctico para la materia de **Programación III - Prof. a cargo (Raul Politi)**.  
La aplicación consiste en una interfaz web interactiva construida utilizando **HTML**, **CSS**, **JavaScript** y **Bootstrap**, aplicando conceptos fundamentales de POO para la organización y reutilización del código.

---

## 🚀 Tecnologías utilizadas

- **HTML5** → estructura del sitio.
- **CSS3** → estilos personalizados.
- **JavaScript (ES6)** → lógica de la aplicación y programación orientada a objetos.
- **Bootstrap 5** → diseño responsive y componentes visuales.

---


## 🧠 Conceptos de POO aplicados

Durante el desarrollo del trabajo se utilizaron distintos conceptos de programación orientada a objetos:

- Constructores
- Encapsulamiento
- Métodos
- Modularización del código
- Manipulación del DOM mediante objetos

---

## ⚙️ Funcionalidades

- Interacción dinámica con JavaScript.
- Estilos personalizados combinados con Bootstrap.
- Organización del código siguiendo buenas prácticas.

---

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/usuario/proyecto-poo.git
```

2. Abrir la carpeta del proyecto.

3. Ejecutar el archivo `index.html` en el navegador.
